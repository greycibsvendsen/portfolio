Girls Who Code

Study Case

Girls Who Code (GWC) is a nonprofit organization that wants to close the gender gap in technology and computer science. They want to inspire more girls to become computer scientists and engineers around the world.

The goal with this project is to introduce young girls to coding programs and make coding educations attract.

Logo

The logo should reflect technology while Robota should be the main character who inspires young girls to code.
The robot symbolizes creativity and strength.
